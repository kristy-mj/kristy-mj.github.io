JavaScript Data Types

Strings - a sequence of characters e.g. "I like cheese"
Numbers - numeric values e.g. 100
Booleans - value of either true or false
Undefined Values - values that are undefined
Null - no value at all
Objects - values that are arbitary collections of properties - these can be added to and removed whenever e.g. {type:"gouda", color:"yellow", taste:"great"}
Arrays

Arrays

An array is a data type specifically for storing sequences of values. It is written as a list of values between square brackets, separated by commas. An array can hold more than one value at a time and you can access the values by referring to an index number (the first value in an array has an index of 0) e.g. [4, 7, 8]
