JavaScript - Scope

JavaScript uses scope chains to set the scope for a given function. There are two types of scope - global scope and local scope.

Global Scope

Variables and functions defined outside a function have global scope. They are accessible anywhere once they've been declared, even inside new functions.

Local Scope

Variables defined inside a function have local scope. These variables become local to that function and cannot be accessed outside that function.

In JavaScript, variables are read starting from the innermost scope then moving outwards. This means the variables are evaluated as if they were declared at the beginning of the scope they exist in.
