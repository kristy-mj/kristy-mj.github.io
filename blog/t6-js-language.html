JavaScript Language

JavaScript Cases

Parentheses ()

  • Used to invoke/execute functions.
  • Used to set limits of if, for and while statements.
  • Used to define function parameters.
  • Used to order expressions such as in maths calculations.

    • brackets []

  • Used to define arrays.
  • Used to access object properties.

    • braces {}

  • Used to define objects.

    • Single Quotes ''

  • Text is placed inside single quotes to create strings.

    • Double quotes ""

  • Same as single quotes - the text inside double quotes are also strings. It is best to stick to one or the other. However some linters and programs seem to have different preferences.
